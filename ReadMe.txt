COMMAND FOR COMPILE AND RUN PROJECT
-----------------------------------
The program was developed using eclipse JAVA Neon.

The program is executed using run configurations in eclipse.

Naive Bayes 

Argument 0 -  Data (test + train folders + SpamWords.txt)
Argument 1 - yes/no ( Yes - Remove Stop word ; No - Do not remove Stop words)


example : Data yes 

----------------------------------------------------------------
Logistic Regression:

Argument 0 : Data (test + train folders + SpamWords.txt)
Argument  1:  yes/no ( Yes - Remove Stop word ; No - Do not remove Stop words)
Argument 2: LearningRate_eta
Argument 3: Lambda
Argument 4: numOfiterations 

example : Data yes 0.01 0.5 500

----------------------------------------------------------------

Please note in case of Logistic regression, the program take lot of time to complete execution.
